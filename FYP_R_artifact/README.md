# FYP Artifact Code

This folder contains the computational artifact files used in this meta-analysis project.

The project investigates the pollination contribution of western and eastern honeybees to temperate flowering plants. The workflow includes literature retrieval, AI-assisted literature screening, manual full-text reading and data extraction, followed by downstream analysis and visualization in R.

## Project workflow

The computational workflow contains four main parts:

1. Literature retrieval using a PubMed crawler.
2. AI-assisted preliminary screening of retrieved literature.
3. Manual full-text reading and data extraction.
4. R-based analysis and visualization, including phylogenetic tree generation and sampling location mapping.

## Description of folders and files

### `crawler/`

This folder contains the Python notebook used for literature retrieval.

File:

```text
crawler/crawler.ipynb
```

The notebook was used to crawl PubMed literature based on search keywords related to the meta-analysis topic. Approximately 10,000 literature records were retrieved before screening.

### `AI traning/`

This folder contains the documentation of the AI-assisted literature screening process.

File:

```text
AI traning/AI Training for Assisted Literature Screening.md
```

This Markdown file records the training and prompting process used to guide ChatGPT in assisting with preliminary literature screening. The purpose of this step was to remove clearly irrelevant papers before manual full-text reading.

The AI-assisted screening was used only as an initial filtering step. The final inclusion of studies and data extraction were completed through manual reading and checking.

### `scripts/`

This folder contains the R scripts used for downstream analysis and visualization.

#### `scripts/01_generate_phylogenetic_tree.R`

This script generates a phylogenetic tree for the plant species included in the project.

The script uses the OpenTree database through the `rotl` package and saves the output tree file as:

```text
results/tree/plants_opentree_clean.tree
```

This script requires internet access because it retrieves taxonomic and phylogenetic information from OpenTree.

#### `scripts/02_generate_sampling_map.R`

This script generates a global map showing the sampling locations used in the project.

The script reads the sampling coordinate file:

```text
data/site.xlsx
```

It uses the icon images stored in:

```text
assets/pin_red.png
assets/pin_blue.png
```

The output files are saved as:

```text
results/map/global_sampling_map.pdf
results/map/global_sampling_map.png
```

### `data/`

This folder contains input data used by the R scripts.

File:

```text
data/site.xlsx
```

The Excel file contains sampling location information. It should include at least the following columns:

```text
lon
lat
group
```

Column descriptions:

- `lon`: longitude of the sampling location.
- `lat`: latitude of the sampling location.
- `group`: group label used to assign different map icons.

### `assets/`

This folder contains image assets used by the map plotting script.

Files:

```text
assets/pin_red.png
assets/pin_blue.png
```

These icons are used to display different groups of sampling locations on the global map.

### `results/`

This folder contains the output files generated by the R scripts.

Phylogenetic tree output:

```text
results/tree/plants_opentree_clean.tree
```

Map outputs:

```text
results/map/global_sampling_map.pdf
results/map/global_sampling_map.png
```

## How to run the R scripts

Please open RStudio and set the working directory to the `FYP_R_artifact` folder.

Then run the following commands in the R console:

```r
source("scripts/01_generate_phylogenetic_tree.R")
source("scripts/02_generate_sampling_map.R")
```

Alternatively, each `.R` script can be opened and run directly in RStudio.

## Important note about paths

The scripts are intended to be run from the `FYP_R_artifact` folder.

If the folder is moved to another computer or another location, please make sure the working directory is set to the main `FYP_R_artifact` folder before running the scripts.

## Expected outputs

After running the scripts, the expected outputs are:
